linux java版本 Hello World

  • 验证java环境安装情况

新建 main.java 文件，注意class类型与文件名保持一致

public class main {
        public static void main(String[] args) {
                System.out.println("hello java " + args[0]);
        }
}

编译main.java

javac main.java

javac 后面直接输入待编译的java文件

编译生成 class 文件

ls

main.class  main.java

运行 java 程序

java main 2021

hello java 2021

注意运行程序时不需要加是 .class 后缀
